# Backend assessment

This repository contains separate deliverables for each part of the assessment. Each folder has its own README with run instructions and details.

| Part | Folder | Summary |
|------|--------|---------|
| **A** | [`A/`](A/) | Python — “Mystic Waves” (`mystic_waves.py`) |
| **B** | [`B/`](B/) | Python — “CargoCraft Fleet” (`cargocraft_fleet.py`) |
| **C** | [`C/`](C/) | Written answers — Xero Accounting API (`task-c.md`) |
| **Frontend** | [`frontend/`](frontend/) | React + TypeScript product detail page (CRA) |

## Quick start

- **A:** `cd A && python3 mystic_waves.py` (see [`A/README.md`](A/README.md))
- **B:** `cd B && python3 cargocraft_fleet.py` (see [`B/README.md`](B/README.md))
- **C:** open [`C/task-c.md`](C/task-c.md)
- **Frontend:** `cd frontend && npm install && npm start` (see [`frontend/README.md`](frontend/README.md))
